## Customization Guide

### How to Customize Your Pinned Message:

1. **Choose a Template:** Pick the template that best fits your server's style
2. **Modify Examples:** Replace example titles with popular/relevant media
3. **Adjust Keywords:** If you changed default keywords in Notifiarr, update them here
4. **Add/Remove Sections:** Include only the media types you have enabled
5. **Personalize Tone:** Adjust formality based on your community
6. **Add Rules:** Include any specific request rules or limitations

### Customization Options:

**Channel Restriction Notice:**
Add this note to your pinned message if you want to remind users:
```
⚠️ IMPORTANT: All media request commands must be used in the #requests channel only!
Commands used in other channels will be rejected.
```

**Keywords:**
Default commands (simplified):
- `!movie` - Add a movie to Radarr
- `!show` - Add a TV show to Sonarr
- `!book` - Add a book to Readarr
- `!audiobook` - Add an audiobook to Readarr

**Additional Media Types:**
If you have Lidarr enabled, add:
```
🎵 Music (Lidarr):
add music [artist or album]
Example: add music Pink Floyd
```

**Server Rules:**
Add any specific guidelines:
```
📋 RULES:
• Maximum 5 requests per day
• No duplicate requests
• Requests are permanent - choose wisely!
```

**Approval Status:**
If NOT using auto-approval, change:
```
❌ All requests require approval! 🎉
→
⏳ Requests require admin approval
```

### Formatting Tips for Discord:

- Use code blocks for commands: \`add movie title\`
- Use bold for emphasis: \*\*IMPORTANT\*\*
- Use headers: \# HEADER or \## Smaller Header
- Use emojis to break up text and add visual interest
- Use horizontal rules for sections: \-\-\-
- Keep line length reasonable (mobile users)

### Copy-Paste Instructions:

1. Choose your preferred template
2. Copy the entire content between the ``` markers
3. Paste into Discord
4. Make any final adjustments
5. Pin the message in your #media-requests channel
